npm配置

发表于 2016-05-26   |  

npm help install

Packages are dropped into the node_modules folder under the prefix. Global installs on Unix systems go to {prefix}/lib/node_modules odules. Global installs on Windows go to {prefix}/node_modules(that is,no lib folder)

npm config ls -l

可列出npm所有的配置

查看某项配置

例如查看prefix配置 npm config ls -l | grep prefix -i or npm config get prefix

修改prefix配置 npm config set prefix "/local"

类Unix系统，

例如OSX 默认全局安装包在/usr/local/lib/node_modules下，node二进制程序在/usr/local/bin/node下

linux下默认全局安装包在/usr/lib/node_modules下，node二进制程序在/usr/bin/nodejs
