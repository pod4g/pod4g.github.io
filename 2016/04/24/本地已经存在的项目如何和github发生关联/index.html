本地已经存在的项目如何和github发生关联

发表于 2016-04-23   |  

  1. 切换到本地项目地址 git init 初始化项目。该步骤会创建一个 .git 文件夹是附属于该仓库的工作树。

  2. 把修改的文件存入暂存区 git add .

  3. 生成一次提交 git commit -m ‘initial commit’

  4. 与远程仓库发生关联 git remote add origin git@github.com:pod4g/sku.git 或者 git remote add origin https://github.com/pod4g/sku

    可能需要执行git config –global push.default matching命令，使得push命令默认push到github的同名仓库中

  5. 把本地项目push到远程github仓库 push -u origin master

本地仓库与远程仓库解除关联rm -rf .git删除该仓库的工作树即可
第2、3步可以在第4步之后执行
