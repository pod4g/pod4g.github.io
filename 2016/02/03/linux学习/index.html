Linux学习笔记

一、linux（MAC）上各种快捷键

  1. vi翻滚命令

    • j & Ctrl+n 向下滚动一行

    • k & Ctrl+p 向上滚动一行 Ctrl+p

      • h 向左移动一个字符

      • l 向右移动一个字符

      • ctrl + f(front) 向下滚动一屏

      • ctrl + b(back) 向上滚动一屏

      • ctrl + d(down) 向下滚动半屏

      • ctrl + u(up) 向上滚动半屏

      • ctrl +

  1. ctrl + a 切换到命令行开始

  2. ctrl + e 切换到命令行结尾

  3. ctrl + l 清屏

  4. ctrl + u 删除光标位置到开头的所有字符

  5. ctrl + k 删除光标位置到结尾的所有字符

  6. ctrl + y 粘贴刚才删除的字符（ctrl＋ｕ和 ctrl+k删除的）

  7. ctrl + c 终止当前命令

  8. ctrl + d 退出shell

  9. ctrl + z 转入后台运行

  10. !! 重复执行刚才最后一条命令

  11. history列出所有历史命令

  12. !number 执行第number条历史命令
